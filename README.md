# Portfolio (React)

Modern personal portfolio - TypeScript, React, Vite, Tailwind.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Routes

- `/about` - introduction and timeline
- `/cv` - curriculum (browser print to PDF)
- `/projects` - project grid

Language toggle: PT / EN (top right).
